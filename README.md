# Klub Sportowy Kraków Południe - Website

Modern, responsive website for Klub Sportowy Kraków Południe sports club.

## Features

- **Home**: Hero section with club branding
- **About**: Club mission, history, and values
- **Classes & Training**: Complete schedule with activity types
- **Coaches/Team**: Staff profiles and expertise
- **Contact**: Contact form and location map
- **Membership**: Join the club information
- **News**: Blog and announcements

## Tech Stack

- React 18
- Vite
- Responsive CSS
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

The site will open at http://localhost:3000

## Build for Production

```bash
npm run build
```
